# 🔥 Landi Scripts – Cinematic Crafting System

## 📦 Description

**Landi Scripts** is an advanced **cinematic crafting system for FiveM** designed for high-quality roleplay servers.

It provides a fully immersive crafting experience with:

* 🎬 Cinematic camera angles
* 🎭 Realistic player animations
* 🧱 Crafting bench with props
* 🔓 XP-based weapon unlock system
* ⏳ Live countdown crafting timer
* 💻 Modern and clean UI (NUI)

Compatible with:

* ESX
* QBCore
* ox_inventory (optional)

## 🧠 How It Works

* You start with **Pistol unlocked**
* After **10 crafts → SMG unlocks**
* After **20 crafts → AK unlocks**

Each craft:

* Takes **5 seconds**
* Shows a **live countdown**
* Gives **XP progression**
* Plays animation + cinematic camera

## 🛠 Installation

1. Place the folder into:

```
resources/[local]/landi_scripts
```

2. Add to your `server.cfg`:

```
ensure landi_scripts
```

3. Restart your server
